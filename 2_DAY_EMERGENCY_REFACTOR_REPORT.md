## 🚀 使用指南

### 快速开始
```bash
# 1. 启动系统 (3-5秒)
cd /root/.openclaw/workspace
node start.js

# 2. 查看系统状态
node start.js --status

# 3. 使用代码生成
node modules/code-generation/skills/code-generation/start-code-generation.js \
  --free-code "创建一个React用户管理组件"

# 4. 管理任务
node modules/task-management/task_manager.js report
node modules/task-management/task_manager.js add "新任务" "任务描述"
```

### 高级功能
```bash
# 1. 使用$team模式代码审查
node modules/code-generation/skills/code-generation/team-mode.js \
  "$(cat your_code.js)" --task="生产代码审查" --models=4

# 2. 执行架构师验证
node modules/code-generation/skills/code-generation/architect-validation.js \
  "$(cat your_code.js)" --level=production --strict

# 3. 运行Oh-my-Codex工作流
node modules/code-generation/skills/code-generation/omc-workflow.js \
  "开发一个完整的Web应用" --detailed --output=project_report.json

# 4. 使用$ri模式持续优化
node modules/code-generation/skills/code-generation/ri-mode.js \
  "优化算法性能" --code="$(cat algorithm.js)" --iterations=10
```

### 系统管理
```bash
# 1. 查看性能监控
node tools/performance-monitor.js --report

# 2. 清理系统缓存
node tools/cache.js --clear

# 3. 重载配置
node tools/config-loader.js --reload

# 4. 查看系统日志
tail -f logs/system.log
```
