## 🛡️ 故障处理流程

### 一级故障：路由调用失败
```bash
# 1. 检查配置文件
cat omc-production-config.json | jq . 2>/dev/null || cat omc-production-config.json

# 2. 检查路由适配器
node -c modules/code-generation/skills/code-generation/omc-router-adapter.js

# 3. 运行诊断测试
node modules/code-generation/skills/code-generation/omc-router-adapter.js test
```

### 二级故障：工作流阶段失败
1. **查看错误详情**: 检查错误日志获取具体错误信息
2. **阶段重试**: 调整阶段参数后重试
3. **降级策略**: 切换为更稳定的路由策略

### 三级故障：系统级故障
1. **启用备份系统**: 切换到备份的工作流
2. **执行回滚**: 使用备份目录恢复系统
3. **故障诊断**: 分析日志，定位根本原因
