# OpenClaw LLM 路由系统

## 概述
为 OpenClaw 设计的多提供商 AI 路由系统，支持原生调用、智能缓存和统一 API 管理。

## 核心特性
1. **多提供商支持**：同时接入 4Sapi、OpenRouter、阿里百炼等主流 AI 服务
2. **智能缓存**：Redis 缓存重复请求，节省流量和成本
3. **高可用**：5-Key 轮询池，自动故障转移
4. **轻量级**：针对 2核4GB 环境优化
5. **易于迁移**：容器化部署，支持迁移到其他服务器

## 架构设计
```
┌─────────────┐
│  客户端请求   │
└──────┬──────┘
       │
┌──────▼──────┐
│ 统一路由网关  │
└──────┬──────┘
       │
┌──────▼──────────────────┐
│     缓存层 (Redis)       │
│   • 请求去重             │
│   • 结果缓存             │
└──────┬──────────────────┘
       │
┌──────▼─────────────┐
│   提供商路由策略     │
│   • 优先级选择       │
│   • 负载均衡         │
└──────┬─────────────┘
       │
┌──────▼──────┬──────┬──────┐
│  4Sapi      │ Open│ 阿里  │
│   NewAPI    │ Router │ 百炼 │
└─────────────┴──────┴──────┘
```

## 快速开始
```bash
# 克隆项目
cd /root/.openclaw/workspace

# 启动服务
python llm_router/main.py
```

## 配置
编辑 `config/settings.yaml` 配置提供商 API Key 和缓存策略。
