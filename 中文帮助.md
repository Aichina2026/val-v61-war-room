# OpenClaw 中文增强系统 - 帮助文档

## 📖 系统简介

**OpenClaw 中文增强系统**是一个完全中文界面的AI系统集成平台，集成了多个先进的AI工具和系统，提供从代码生成到架构设计的完整解决方案。

## 🚀 快速开始

### 1. 启动中文系统

```bash
# 启动完整中文系统
node 中文系统启动器.js "你的项目主题"

# 示例：设计电商系统
node 中文系统启动器.js "设计一个高性能电商网站"

# 启动服务并指定AI模型数量
node 中文系统启动器.js "优化现有系统性能" --模型数=3 --启动服务
```

### 2. 启动4SAPI辩证系统

```bash
# 仅启动辩证辩论系统
node 4sapi_dialectic_multi_ai_system.js "辩论主题"

# 示例：架构选择辩论
node 4sapi_dialectic_multi_ai_system.js "微服务架构 vs 单体架构" --models=5
```

### 3. 启动代码生成系统

```bash
# 使用集成工作流
node modules/code-generation/skills/code-generation/start-code-generation.js --integrated "你的需求"

# 单独使用各组件
node modules/code-generation/skills/code-generation/omc-workflow.js "需求描述"
node modules/code-generation/skills/code-generation/team-mode.js "代码内容" --task="审查任务"
```

## 🔧 系统功能

### 1. 4SAPI辩证分析
- **战略层(S1)**：长期目标分析
- **结构层(S2)**：系统架构设计
- **系统层(S3)**：交互关系规划
- **服务层(S4)**：API接口定义
- **算法层(A)**：核心算法设计
- **平台层(P)**：部署架构规划
- **接口层(I)**：用户体验设计

### 2. 多AI模型辩论
- **5个中文AI专家**：架构师、安全专家、性能工程师、质量保证、产品经理
- **3轮辩证辩论**：命题→反驳→共识
- **零错误验证**：生产级质量保证

### 3. 代码生成系统
- **free-code集成**：Claude code纯洁版
- **Oh-my-Codex工作流**：端到端编排
- **$team模式**：4模型并行审查
- **$ri模式**：持久执行循环
- **架构师验证**：生产级架构验证

### 4. 集成服务
- **HTTP服务**：端口3000
- **RESTful API**：完整API支持
- **控制面板**：Web界面管理
- **实时监控**：系统状态监控

## ⚙️ 配置选项

### 中文系统启动器选项
- `--模型数=5`：指定AI模型数量（默认：5）
- `--启动服务`：启动集成HTTP服务
- `--帮助`：显示帮助信息

### 4SAPI辩证系统选项
- `--models=5`：AI模型数量（默认：5）
- `--mode=zero-error`：辩证模式
- `--output=路径`：保存结果目录

### 代码生成系统选项
- `--template=组件`：代码模板类型
- `--language=javascript`：编程语言
- `--framework=react`：框架类型
- `--quality=production`：质量等级

## 🔍 故障排除

### 常见问题

1. **系统启动失败**
   - 检查文件权限：`chmod +x *.js`
   - 检查Node.js版本：`node --version`
   - 检查依赖：`npm list`

2. **服务无法访问**
   - 检查端口占用：`netstat -tlnp | grep 3000`
   - 检查防火墙：`ufw status`
   - 重启服务：`pkill -f "node.*3000"`

3. **中文显示异常**
   - 检查系统编码：`echo $LANG`
   - 安装中文字体：`apt-get install fonts-wqy-zenhei`
   - 设置环境变量：`export LANG=zh_CN.UTF-8`

### 错误代码
- **ERR_001**：文件不存在
- **ERR_002**：权限不足
- **ERR_003**：服务端口占用
- **ERR_004**：网络连接失败
- **ERR_005**：配置错误
